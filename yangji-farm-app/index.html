    手機安裝

    把養肌農場加入手機主畫面

    Android / Chrome

    1. 用 Chrome 開啟正式網址。
    2. 按「安裝到手機」或瀏覽器選單。
    3. 選擇「安裝應用程式」或「加入主畫面」。
    4. 主畫面會出現養肌農場圖示。

    iPhone / Safari

    1. 用 Safari 開啟正式網址。
    2. 按下方分享按鈕。
    3. 選擇「加入主畫面」。
    4. 確認名稱為「養肌農場」後加入。
    正式安裝需要 HTTPS 網址。目前本機網址可測試畫面；要讓多支手機穩定安裝，下一步要部署到 HTTPS。

    關於養肌農場

    把實證運動變成每日農場任務

    適合長者

    使用大按鈕、清楚文字與語音說明，任務以坐站、步行、扶椅平衡與蛋白質紀錄為主。

    實證基礎

    依據你的運動處方資料，將阻力訓練、平衡訓練、有氧活動與營養協同轉成簡單任務。

    安全優先

    若有胸悶、頭暈、明顯疼痛、剛跌倒或醫師限制運動，請先停止並詢問專業人員。
